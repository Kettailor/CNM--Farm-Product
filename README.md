# Farm Product Traceability System

## Projects

- Backend (NestJS + Prisma): `backend/`
- Frontend (Next.js + React + TailwindCSS): `frontend/`

## Run Backend

```bash
cd backend
cp .env.example .env
npm install
npm run prisma:generate
npm run dev
```

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```
