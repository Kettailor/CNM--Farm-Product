# Changelog

Tất cả thay đổi đáng chú ý của dự án sẽ được ghi lại tại đây.

Định dạng theo SemVer: `MAJOR.MINOR.PATCH`.

## [1.0.0] - 2026-06-03

### Added

- Hoàn thiện nền tảng KetKat-EcoFarm cho quản lý nông trại và truy xuất sản phẩm.
- Dashboard quản lý trang trại, khu vực, bản đồ, vật nuôi, công việc, kho, hồ sơ hóa chất và chứng từ.
- Truy xuất QR public cho vật nuôi và bản đồ trang trại public.
- PostgreSQL schema chuẩn trong `database/FarmHub_schema.sql`.
- Docker Compose cho app, PostgreSQL, Adminer và Nginx.
- README đầy đủ, GitHub issue templates, PR template, Dependabot và GitHub Actions CI/CD.

### Release

- Tag đầu tiên: `v1.0.0`.
